# Job Eligibility Checker

## What it does
Takes a set of candidate details as input (things like age, qualifications, or experience) and checks them against a set of eligibility rules, then returns whether the candidate qualifies.

## How it works
1. Collect candidate information through input prompts.
2. Run each piece of information through the relevant eligibility rule.
3. Combine the results using logical operators.
4. Print a clear pass/fail result, ideally with a reason if it fails.

## Why I built it
Practice combining multiple conditions with logical operators (`and`, `or`, `not`) instead of relying on a single `if` statement.

## Skills used
Variables, user input, logical operators, nested conditionals, clean output formatting.
